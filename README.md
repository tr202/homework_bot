# homework_bot
python telegram bot
### Технологии
Python 3.9
### Запуск проекта в dev-режиме
- Установите и активируйте виртуальное окружение
- Установите зависимости из файла requirements.txt
```
pip install -r requirements.txt
``` 
- В папке с файлом homework.py выполните команду:
```
python -m homework
```
